## Justification for Permissions

### storage
Required to save user preferences (highlight color, style, enabled/disabled state, theme preference) locally on the device.

### activeTab
Required to highlight medical terms on the current webpage when the user clicks the extension icon.

### contextMenus
Required to provide right-click options for looking up selected text.

### scripting (optional)
Allows the extension to automatically highlight terms on websites without requiring the user to click the icon each time.

### <all_urls> (optional host permission)
Allows the extension to run on all websites. This is optional and the user can choose to grant it for convenience.

---

## Single Purpose Description

This extension serves a single purpose: to help medical students learn terminology by highlighting medical terms on webpages and displaying their definitions. All features (highlighting, definitions, search, customization) directly support this educational purpose.
