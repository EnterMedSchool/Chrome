EMS Medical Glossary Test Page

Open the browser DevTools (F12) and check the Console for messages starting with "[EMS]".

Test 1: Abbreviations (Case-Sensitive)

The following abbreviations should be highlighted when typed in UPPERCASE:

  • ACTH - Adrenocorticotropic hormone
  • DNA - Deoxyribonucleic acid
  • RNA - Ribonucleic acid
  • ATP - Adenosine triphosphate

These should NOT be highlighted (lowercase): acth, dna, rna, atp

Test 2: Full Medical Terms

The following full terms should be highlighted regardless of case:

  • adrenocorticotropic hormone
  • cortisol
  • pituitary gland
  • adrenal cortex

Test 3: Inline Text

The hormone ACTH is released from the anterior pituitary gland. It stimulates the adrenal cortex to produce cortisol. This is part of the hypothalamic-pituitary-adrenal axis.

Debug Instructions
